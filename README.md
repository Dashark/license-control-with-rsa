# license-control-with-rsa
license, validity, RSA, public key, private key, bind device, java | python.软件授权可用设备绑定有效期限制私钥加密公钥解密（逆用，未深究利弊，仅实现仅学习，部分代码为他人博客摘取整合，有删改）
